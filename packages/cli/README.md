> Programmatically create command lines based on [Consola](https://github.com/unjs/consola), powered by [Funish](https://funish.net/).

## Getting started

```bash
# npm
$ npm install @funish/cli

# yarn
$ yarn add @funish/cli

# pnpm
$ pnpm add @funish/cli
```
